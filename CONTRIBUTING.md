# Contributing

## Who can commit

This repository has two **code owners** (see [.github/CODEOWNERS](.github/CODEOWNERS)):

- **Carrie Gorringe** — [@CarrieGorringe](https://github.com/CarrieGorringe)
- **Scott Thurlow** — [@ScottThurlow](https://github.com/ScottThurlow)

### Code owners
May commit directly to the default branch (`main`).

### Everyone else
Must open a **pull request**. PRs require approval from at least one code owner before they can be merged — branch protection enforces a code-owner review on every change.

## Workflow for contributors

1. Fork or branch from `main`.
2. Make your changes on a feature branch.
3. Open a pull request against `main`.
4. Wait for review and approval from a code owner.
5. A code owner merges once approved.
